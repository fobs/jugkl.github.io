--- layout: post title: Stateless Authentication for REST and Microservices with Json Web Token author: Yannick Mortier category : talk ---

The authentication of users of microservices or stateless REST requests is a challenge. JSon Web Token provides solutions for this.
The talk explains the functionality and application of JWT in a real-life example with an authentication server implemented as microservice. Also, security problems of stateless authentication and their solutions are discussed. This talk is in English.

29.11.2017, 19:00, Chaos Computer Club Kaiserslautern

Sprecher:

Yannick Mortier ist als Java Software-Entwickler bei TOPdesk tätig. Aber auch in der Freizeit beschäftigt er sich mit eigenen Projekten in diesem Bereich und verwendet dabei undogmatisch verschiedene Technologien für Backend und Frontend wie Docker, Java-Frameworks wie Spring sowie auch das neue Angular mit TypeScript.
