--- layout: post title: Stateless Authentication für REST und Microservices mit Json Web Token author: Yannick Mortier category : talk ---

Die Authentifizierung von Benutzern gegenüber Microservices oder für wirklich zustandslose REST Anfragen bietet einige Herausforderungen, die JSon Web Token zu lösen versucht.
Dieser Vortrag erklärt die Funktion und Anwendung von JWT anhand eines praktischen Beispiels mit einem als Microservice implementierten Authentifizierungsserver und weist auch auf ein paar Sicherheitsbedenken sowie Lösungsansätze für diese hin.

16.11.2017, 19:00, Chaos Computer Club Kaiserslautern

Sprecher:

Yannick Mortier ist als Java Software-Entwickler bei TOPdesk tätig. Aber auch in der Freizeit beschäftigt er sich mit eigenen Projekten in diesem Bereich und verwendet dabei undogmatisch verschiedene Technologien für Backend und Frontend wie Docker, Java-Frameworks wie Spring sowie auch das neue Angular mit TypeScript.
